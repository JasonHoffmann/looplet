The Looplet Companion plugin builds on the functionality of the Looplet Theme

== Description ==

This plugin does two simple things.

1. It adds the custom post type “Looplet”.
2. It adds dummy posts to the Looplet post type to get you started.

*Note: Any dummy content will be deleted upon deactivation of this plugin*
